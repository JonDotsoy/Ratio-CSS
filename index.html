Ratio CSS

Es un simple framework CSS que facilita la creación de componentes responsivos con relacionada de aspectos. wiki

Ejemplo HTML:

Sin Ratio CSS

<div>
    <img src="http://placehold.it/620x280" alt="">
</div>

Con Ratio CSS

<div class="ratio ratio-16by9">
    <img src="http://placehold.it/620x280" alt="" class="ratio-item">
</div>

Ejemplo en vivo

http://fiddle.jshell.net/44Qkd/4/show/light/
